# 🏋️ PowerFit Gym Website

A dynamic fitness website that generates workout plans, diet recommendations, and transformation timelines based on user input.

---

## 🚀 Features

- Workout Plan Generator
- BMI Calculator
- Diet Recommendation
- Transformation Time Estimation
